# 📝 To-Do List en C

Proyecto de la **Semana 1** de la ruta de estudio de **Desarrollo de Sistemas**.

## 📌 Objetivo
Crear un programa en C que gestione una lista de tareas, con opciones para:
1. Agregar tarea.
2. Listar tareas.
3. Marcar como completada.
4. Guardar en archivo de texto.

## ⚙️ Compilación
```bash
make
./to_do_list
